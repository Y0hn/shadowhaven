# Shadowhaven
## Skolsky Projekt 2023/24

### Goal:
	Vytvorit funkcne singleplayer 2D RPG s:
		combat systemom
			melee
			ranged
			magic
		equipment systemom
		moznostou hrania na ovladaci aj klavesnici
		dandznami (v jaskynach, budovach ...)
			v styte Binding of Issac
		interaktivnimi NPC
